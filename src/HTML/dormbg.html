Dormbg endpoint

The Dormbg endpoint gets info on house types you can buy ingame.

Endpoint:

http://localhost:4000/v1/dormbg

Example request:

fetch('http://localhost:4000/v1/dormbg/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

dormbg[dormbgID].ID
dormbg[dormbgID].assetID
dormbg[dormbgID].type
dormbg[dormbgID].gender
dormbg[dormbgID].data
dormbg[dormbgID].metadata
dormbg[dormbgID].name

Data properties:

dormbg[dormbgID].data.x
dormbg[dormbgID].data.y
dormbg[dormbgID].data.tag
dormbg[dormbgID].data.name
dormbg[dormbgID].data.price
dormbg[dormbgID].data.member
dormbg[dormbgID].data.rarity
dormbg[dormbgID].data.category
dormbg[dormbgID].data.flavorText

Get more specific results using the Dormbg ID in the API endpoint.

http://localhost:4000/v1/dormbg/ID/

Example request:

fetch('http://localhost:4000/v1/dormbg/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":749,"type":"dormbg","gender":0,"data":{"x":800,"y":470,"tag":"suburbs","name":"Starter Home","price":0,"member":0,"rarity":0,"category":"Comfy","flavorText":"A true classic."},"metadata":{"vIcon":2},"name":"Starter Home"}
