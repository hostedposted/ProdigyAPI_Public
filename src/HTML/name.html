Name endpoint

The Name endpoint gives info on all first names you can get ingame.

Endpoint:

http://localhost:4000/v1/name

Example request:

fetch('http://localhost:4000/v1/name/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

name[nameID].ID
name[nameID].assetID
name[nameID].type
name[nameID].gender
name[nameID].data
name[nameID].metadata
name[nameID].name

Data properties:

name[nameID].data.gender
name[nameID].data.name
name[nameID].data.value
name[nameID].data.type

Get more specific results using the Name ID in the API endpoint.

http://localhost:4000/v1/name/ID/

Example request:

fetch('http://localhost:4000/v1/name/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":2212,"type":"name","gender":0,"data":{"gender":2,"name":"Aaron","value":"Aaron","type":0},"metadata":{},"name":"Aaron"}
