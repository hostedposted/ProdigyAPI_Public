FaceColor endpoint

The FaceColor endpoint gives info on all of the skin colors in Prodigy.

Endpoint:

http://localhost:4000/v1/faceColor

Example request:

fetch('http://localhost:4000/v1/faceColor/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

faceColor[faceColorID].ID
faceColor[faceColorID].assetID
faceColor[faceColorID].type
faceColor[faceColorID].gender
faceColor[faceColorID].data
faceColor[faceColorID].metadata
faceColor[faceColorID].name

Data properties:

faceColor[faceColorID].data.price
faceColor[faceColorID].data.colors

Get more specific results using the FaceColor ID in the API endpoint.

http://localhost:4000/v1/faceColor/ID/

Example request:

fetch('http://localhost:4000/v1/faceColor/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":342,"type":"faceColor","gender":0,"data":{"price":50,"colors":[[0,0,0],[110,118,139],[217,224,245],[200,148,102],[187,107,135],[255,127,169]]},"metadata":{},"name":""}
