MathTownInterior endpoint

The MathTownInterior endpoint gives info on the Tower Interiors in Tower Town.

Endpoint:

http://localhost:4000/v1/mathTownInterior

Example request:

fetch('http://localhost:4000/v1/mathTownInterior/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

mathTownInterior[mathTownInteriorID].ID
mathTownInterior[mathTownInteriorID].assetID
mathTownInterior[mathTownInteriorID].type
mathTownInterior[mathTownInteriorID].gender
mathTownInterior[mathTownInteriorID].data
mathTownInterior[mathTownInteriorID].metadata
mathTownInterior[mathTownInteriorID].name

Data properties:

mathTownInterior[mathTownInteriorID].data.name
mathTownInterior[mathTownInteriorID].data.tag
mathTownInterior[mathTownInteriorID].data.description
mathTownInterior[mathTownInteriorID].data.element
mathTownInterior[mathTownInteriorID].data.flavorText
mathTownInterior[mathTownInteriorID].data.occupantLevels

Get more specific results using the MathTownInterior ID in the API endpoint.

http://localhost:4000/v1/mathTownInterior/ID/

Example request:

fetch('http://localhost:4000/v1/mathTownInterior/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":7940,"type":"mathTownInterior","gender":0,"data":{"name":"Magma Rock Kitchen","tag":"tt-interior-fire-1","description":"","element":3,"flavorText":"","occupantLevels":[0,0,1,2,2]},"metadata":{"type":"singleImage","v":2,"vIcon":1},"name":"tt-interior-fire-1"}
