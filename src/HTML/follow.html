Follow endpoint

The Follow endpoint gives info on all of the buddies that you can get in Prodigy, which follow you around.

Endpoint:

http://localhost:4000/v1/follow

Example request:

fetch('http://localhost:4000/v1/follow/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

follow[followID].ID
follow[followID].assetID
follow[followID].type
follow[followID].gender
follow[followID].data
follow[followID].metadata
follow[followID].name

Data properties:

follow[followID].data.tag
follow[followID].data.name
follow[followID].data.price
follow[followID].data.member
follow[followID].data.rarity
follow[followID].data.flavorText

Get more specific results using the Follow ID in the API endpoint.

http://localhost:4000/v1/follow/ID/

Example request:

fetch('http://localhost:4000/v1/follow/0/').then(response => response.json()).then(data => {console.log(data)})
