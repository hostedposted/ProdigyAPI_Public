Boots endpoint

Overview

The boots endpoint gets info on boots in Prodigy, which are found in the player's inventory.

Endpoint:

http://localhost:4000/v1/boots/

Example request:

fetch('http://localhost:4000/v1/boots/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

boots[bootID].ID
boots[bootID].assetID
boots[bootID].type
boots[bootID].gender
boots[bootID].data
boots[bootID].metadata
boots[bootID].name

Data properties:

boots[bootID].data.drop
boots[bootID].data.name
boots[bootID].data.price
boots[bootID].data.member
boots[bootID].data.rarity
boots[bootID].data.effects
boots[bootID].data.flavorText

Get more specific results using the Boot ID in the API endpoint. Get more specific results using the ad ID in the API endpoint.

http://localhost:4000/v1/boots/ID/

Example request:

fetch('http://localhost:4000/v1/boots/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":206,"type":"boots","gender":0,"data":{"drop":0,"name":"Magi Runners","price":100,"member":1,"rarity":1,"effects":[34],"flavorText":"It is said that a Magi once ran to the sun with these shoes on. You're not sure if you could, but they sure are fast."},"metadata":{"vIcon":2},"name":"Magi Runners"}
