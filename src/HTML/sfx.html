Sfx endpoint

The Sfx endpoint gives info on all of the sounds in the game.

Endpoint:

http://localhost:4000/v1/sfx

Example request:

fetch('http://localhost:4000/v1/sfx/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

sfx[sfxID].ID
sfx[sfxID].assetID
sfx[sfxID].type
sfx[sfxID].gender
sfx[sfxID].data
sfx[sfxID].metadata
sfx[sfxID].name

Data properties:

sfx[sfxID].data.filename

Get more specific results using the Sfx ID in the API endpoint.

http://localhost:4000/v1/sfx/ID/

Example request:

fetch('http://localhost:4000/v1/sfx/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":1824,"type":"sfx","gender":0,"data":{"filename":"sfx-main"},"metadata":{"assets":{"default":{"v":2}}},"name":"sfx-main"}
